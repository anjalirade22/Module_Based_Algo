## Quick Start

1. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Configure credentials:**
   Edit `config/credentials.py` with your broker API details

3. **Test connection:**
   ```bash
   python tests/broker_login_test.py
   ```

4. **Run the system:**
   ```bash
   python main.py --mode paper
   ```
